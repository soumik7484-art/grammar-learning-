# 🌌 Cinematic Interactive Portfolio — Soumik Chatterjee

Welcome to the repository of my cinematic, award-winning interactive portfolio built using React, Tailwind CSS, Framer Motion, and GSAP.

🔗 **Live Deployment**: 👉 [https://mcq-livid.vercel.app](https://mcq-livid.vercel.app)

Each section is designed as a distinct high-fidelity immersive environment with custom color coding, responsive SVG animations, cursor tracking, and spring physics.

---

## 🎨 System Highlights & Worlds

1. **Intro Loader (Neon Emerald)**: A black particle-field intro with a glowing, pulsing AI core assembling typography letter-by-letter.
2. **Hero AI Lab (Deep Violet)**: An interactive futuristic AI laboratory showing real-time pulsing metric graphs, flying drones, and a floating profile card featuring a custom avatar.
3. **About Hologram Room (Cyan)**: A holographic projection system where the hologram turns towards your cursor, waves, and smiles when approached.
4. **Medical AI Diagnostics (Blue/Red)**: An AI Doctor examining a patient. Includes concentric eye-tracking, active heartbeat monitor grids, and floating diagnostic telemetry.
5. **Legal AI Detective (Amber)**: A dim, rain-swept investigator desk. The detective raises binoculars to track your cursor, illuminating evidence boards and clue connections.
6. **Prescription OCR (Emerald)**: An active laser scanner. Hovering over a physical medical card digitizes unstructured text into structured digital JSON outputs.
7. **X-Ray Analyzer (Sky Blue)**: A 3D rotatable viewport. Selecting skeletal nodes highlights fractures, displays anomaly heatmaps, and triggers diagnostic labels.
8. **AI Chatbot Robot (Pink)**: A cute floating assistant robot that smiles, waves, and replies to custom queries in real-time.
9. **Skills Command Center (Multi-Color)**: High-energy floating skill holograms (React, Node, MongoDB, Python, TensorFlow) displaying capacity level progress upon cursor interaction.
10. **Direct Transmission Hub (Indigo)**: Custom animated character cards representing communication links (GitHub Cat, LinkedIn Suited Man, Gmail Postman, Instagram Cameraman).

## 🛠️ Stack & Technologies

- **Framework**: [React.js](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations & Kinetics**: [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll**: [Lenis Scroll](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Local Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/soumik7484-art/port.git
   cd port
   ```

2. **Install all dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```

4. **Compile and build for production**:
   ```bash
   npm run build
   ```
